# sigaa-jumper
Login automático no SIGAA e pulo de telas de aviso  

## Como usar

1. Baixe este repositorio e descompacte em algum lugar
2. Entre nas extensões do seu navegador
3. Procure pelo botão "Carregar sem compactação" e clique
4. Escolhe a pasta desejada (sigaa-chromium, sigaa-firefox etc)
5. Pronto, vai aparecer na barra, clique em extensões e pine ela se quiser.
6. Clique nela, clique em configurações, ponha seu usuário e senha do sigaa e salve.  

Atualmente funciona para os SIGAA do:
- IFSC
